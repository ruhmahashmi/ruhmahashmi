## ⭐ Flagship Project — AILA

> **An Intelligent Lecturing Assistant** &nbsp;·&nbsp; [React](https://reactjs.org/) · [Next.js](https://nextjs.org/) · [FastAPI](https://fastapi.tiangolo.com/) · [LlamaIndex](https://www.llamaindex.ai/) · [Gemini](https://deepmind.google/technologies/gemini/) · [SQLite](https://www.sqlite.org/)

Full-stack AI system that **automates high-quality assessment generation** from lecture slides via RAG + Knowledge Graph pipelines. KG-based approach outperforms standard generation by **5% accuracy** across 450+ evaluated MCQs. Funded by Drexel UREP grant.

- 🧠 Multi-stage ETL pipeline modeling prerequisite relationships as a **Directed Acyclic Graph**
- 🔁 Interactive **Human-in-the-Loop** concept map editor for real-time instructor verification
- 📊 Proven: **16% increase in student knowledge retention** (60-student study)
- 📄 3× peer-reviewed publications, including **SIGCSE 2026** (accepted)

[![AILA Repo](https://img.shields.io/badge/View_AILA_on_GitHub-181717?style=for-the-badge&logo=github)](https://github.com/ruhmahashmi/AILA)
